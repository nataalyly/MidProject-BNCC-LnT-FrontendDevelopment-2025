List Course

UI/UX Design

Kelas UI/UX Design mengajarkan cara merancang antarmuka yang menarik, fungsional, dan responsif dengan prinsip desain visual serta penelitian pengguna. Peserta juga akan belajar menggunakan alat seperti Figma atau Adobe XD untuk membuat wireframe dan prototipe. Kelas akan diadakan sekali dalam seminggu, hari dan jam sesuai kesepakatan bersama.

Front-End Development

Kelas Front-end Development mengajarkan pembuatan antarmuka web atau aplikasi dengan HTML, CSS, dan JavaScript. Peserta juga akan mempelajari framework seperti React, Angular, atau Vue untuk meningkatkan efisiensi pengembangan. Kelas akan diadakan sekali dalam seminggu, hari dan jam sesuai kesepakatan bersama.

Back-End Development

Kelas Back-end Development mengajarkan pembangunan server, database, dan fungsionalitas web dengan bahasa seperti Python, Ruby, atau PHP. Peserta juga akan mempelajari framework, pengelolaan basis data, API, keamanan, dan skalabilitas sistem. Kelas akan diadakan sekali dalam seminggu, hari dan jam sesuai kesepakatan bersama.

Java Programming

Kelas Java Programming mengajarkan dasar-dasar Java untuk pengembangan aplikasi mobile, desktop, dan web. Peserta akan mempelajari alur program, struktur data, OOP, serta penggunaan pustaka dan framework. Kelas akan diadakan sekali dalam seminggu, hari dan jam sesuai kesepakatan bersama.

C Programming

Kelas C Programming mengajarkan dasar-dasar bahasa C, termasuk struktur data, manajemen memori, dan alokasi dinamis. Peserta akan mempelajari cara membuat aplikasi sistem yang efisien dan cepat. Kelas akan diadakan sekali dalam seminggu, hari dan jam sesuai kesepakatan bersama.
